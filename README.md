# Chart.js Dashboard

This project is a dashboard created using Chart.js library to visualize data in various chart formats.

## Technologies Used

- React.js
- Chart.js
- CSS

## Features

- Bar charts
- Radar charts
- Pie charts
- Scatter plots
- Polar area charts
- Area charts
- Bubble charts
- Toggle background color functionality
